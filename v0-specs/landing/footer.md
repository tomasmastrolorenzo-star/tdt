# Component Specification: Footer

**Priority**: Low  
**Status**: ⏳ Pending

## 🤖 V0 Prompt

```
Create a footer component for a landing page using Next.js 14, TypeScript, Tailwind CSS.

COMPONENT PURPOSE:
Standard footer with links, legal info, social media, and branding.

DESIGN: Dark background, multi-column layout

KEY FEATURES:
1. Logo and tagline
2. Link columns (Services, Company, Legal, Support)
3. Social media icons
4. Copyright notice
5. Newsletter signup (optional)

LAYOUT:
- 4-column grid on desktop
- Stacked on mobile
- Bottom bar with copyright

STYLING:
- Background: bg-slate-900
- Text: text-slate-400
- Links: hover:text-white
- Divider: border-slate-800

EXAMPLE LINKS:
- Services: Instagram, TikTok, Pricing
- Company: About, Blog, Careers
- Legal: Terms, Privacy, Refunds
- Support: FAQ, Contact, Help Center
```
